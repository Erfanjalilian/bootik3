# پنل مدیریتی - راهنمای استفاده

## دسترسی به پنل مدیریتی

**آدرس:** `http://localhost:3000/admin1383`

**رمز عبور:** `1383`

---

## ویژگی‌های پنل

### 1. **داشبورد** 📊
- نمایش آمار کلی: تعداد محصولات، بنرها، کاربران و سفارشات
- دسترسی سریع به بخش‌های مختلف

### 2. **مدیریت محصولات** 📦

**امکانات:**
- ✅ افزودن محصول جدید
- ✅ ویرایش محصولات موجود
- ✅ حذف محصولات
- ✅ نمایش لیست تمام محصولات

**فیلدهای محصول:**
- نام، توضیحات، قیمت، قیمت اصلی
- تخفیف (%)، موجودی، امتیاز
- **لینک‌های عکس** (بدون آپلود فایل - فقط URL)
- سایز‌ها، رنگ‌ها، مشخصات
- برند، دسته‌بندی
- حالت‌ها: پرفروش، جدید، تخفیف دار

**API Endpoint:**
```
POST   /api/admin/products     - افزودن محصول
PUT    /api/admin/products     - ویرایش محصول
DELETE /api/admin/products?id=X - حذف محصول
GET    /api/admin/products     - دریافت لیست محصولات
```

### 3. **مدیریت بنرها** 🖼️

**امکانات:**
- ✅ ایجاد بنر جدید (بنر اصلی یا تبلیغ)
- ✅ ویرایش بنرهای موجود
- ✅ حذف بنرها
- ✅ نمایش پیش‌نمایش بنر
- ✅ بنرها به‌صورت خودکار در صفحه اصلی به‌روز می‌شوند

**فیلدهای بنر:**
- عنوان، زیرعنوان
- **لینک عکس** (URL بدون آپلود)
- لینک هدف (مسیر داخلی سایت)
- نوع: بنر اصلی یا تبلیغ

**API Endpoint:**
```
POST   /api/admin/banners      - افزودن بنر
PUT    /api/admin/banners      - ویرایش بنر
DELETE /api/admin/banners?id=X - حذف بنر
GET    /api/admin/banners      - دریافت لیست بنرها
```

### 4. **مدیریت کاربران** 👥

- ✅ نمایش لیست تمام کاربران ثبت‌نام شده
- ✅ نمایش شماره تلفن، تاریخ عضویت، آخرین بازدید

**API Endpoint:**
```
GET /api/admin/users - دریافت لیست کاربران
```

### 5. **مدیریت سفارشات** 📋

**وضعیت:** در حال توسعه

این بخش هنوز کامل نشده است. ویژگی‌های آتی:
- نمایش فهرست سفارشات
- ویرایش وضعیت سفارش
- مشاهده جزئیات سفارش

### 6. **تنظیمات سایت** ⚙️

**امکانات:**
- ✅ ویرایش اطلاعات سایت (نام، شعار، تلفن، آدرس، ایمیل)
- ✅ مدیریت صفحه "درباره ما"
  - عنوان، زیرعنوان، داستان
  - ماموریت و چشم‌انداز
- ✅ مدیریت شبکه‌های اجتماعی
  - اینستاگرام، تلگرام، واتس‌اپ

**API Endpoint:**
```
GET /api/admin/settings  - دریافت تنظیمات
PUT /api/admin/settings  - ویرایش تنظیمات
```

## تغییرات در صفحه اصلی

بنرهای صفحه اصلی اکنون از API پویا می‌آیند. هنگامی که مدیر بنر را ویرایش یا حذف کند، صفحه اصلی به‌صورت خودکار به‌روز می‌شود.

**فایل:** `components/home/BannerSection.tsx`

## نکات مهم

### 🔐 امنیت
- پنل مدیریتی از رمز عبور ساده محافظت می‌شود
- **توجه:** در محیط production باید از احراز هویت محکم‌تر (JWT، OAuth) استفاده شود

### 🖼️ مدیریت عکس‌ها
- **نیز برای محصولات نیز برای بنرها**
- بجای آپلود فایل، فقط لینک عکس قابل قبول است
- عکس‌ها می‌توانند از CDN یا سرور خارجی باشند

- تمامی داده‌ها در فایل‌های JSON در `/data/` ذخیره می‌شوند
- کپی پشتیبانی برای فایل‌های مهم توصیه می‌شود
